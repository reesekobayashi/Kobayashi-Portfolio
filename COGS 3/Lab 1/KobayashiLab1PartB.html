HTML Terms and Definitions

What is a DOCTYPE Statement?

  • It is an instruction to the web browser about what version of HTML is written in.
  • It defines that this document is an HTML document

How to Use ordered, unordered, and description lists

Ordered lists:

  1. are numbered
  2. has the letters "ol" for the 'ordered list' tag
  3. ordered list elements contain list tags labled "li"

Unordered lists:

  • are bulleted points
  • has the letters "ul" for the unordered list tag
  • unordered list elements contain list tags aslo labled "li"

Description lists

Have dl, dt, and dd elements

dl
dl is the elements that makes the description list
dt
dt is the text that will be described
dd
dd is a text that describes what was written in dt

The Difference Between Text Editor and Word Processor:

  1. a text editor used for html are used to create and modify web pages

What is the Client-Server Model?

It is a distributed computing model that segregates information users(clients) from information providers(servers).
